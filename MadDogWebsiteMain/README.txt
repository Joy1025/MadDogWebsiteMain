#STEPS:
1. Install Visual Studio. You can download Enterprise with your ASU student ID @ https://webapp4.asu.edu/eacademy-sso/authn.
	if you decided not to get the Enterprise edition, the free version will also work.
2. Open VS in 'Tools', select 'Extensions and Updates' this is where you can customize your IDE to your taste; however, download the Github extension.
	a. Select 'online' to browse for extensions 
	b. Search & install Github for VS extension.
3. Restart VS and your Github repositories will be connected to your IDE.
4. Select Github from the homepage and clone the repository into any folder you desire to work on. (This will create a local copy from the Github server into your machine)
5. You should now be able to open the project.
